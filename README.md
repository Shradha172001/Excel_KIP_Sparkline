1. Sparkline
   A sparkline is a small chart inside a single Excel cell that shows a trend or pattern in data.

Why use Sparklines?

Quick trend analysis
Saves space
Easy comparison across rows

2. KPI (Key Performance Indicator)
   A KPI is a measurable value that shows how well a process or business is performing against its goals.

Why use KPIs?

Measure performance
Identify problems quickly
Support decision-making

3. Protection (Protect Sheet / Workbook)
   Protection is used to prevent accidental changes to important data or formulas

Why use Protection?

Avoid formula deletion
Maintain data accuracy
Control user access
